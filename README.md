# Web Ticket

Sistema para controle de atendimento em laboratórios médicos.

## Descrição

Sistema de tickets com três tipos de senha:
- SP - Senha Prioritária (15 min)
- SG - Senha Geral (5 min)  
- SE - Senha para Exames (1 min)

## Funcionalidades

- Emissão de senhas pelo cliente
- Chamada de senhas pelo atendente
- Painel de visualização do sistema
- Relatórios diários

## Como usar

Abra o arquivo `index.html` no navegador.
